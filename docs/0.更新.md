# Version

## 0.1.3 修改

- 终端时间戳小数位数: 3 -> 5
- 终端时间戳长度修改为: 13(纯时间戳) + 3(中括号以及空格)
- 修改`SSH.exec_run()`方法，加入`sudo`、`password`参数，支持提权运行
- 添加`SSH.exec_run_sudo()`方法，支持提权运行，是`SSH.exec_run()`方法的简单包装
- `CmdRecord`类添加`stdout`、`stderr`属性，并在执行run方法时存储
- 提取`stdout`处理，添加到`SSH._cmd_output_process()`内部方法
- 添加`SSH.close_sub_channels`、`SSH.close_sub_sftp()`、`SSH.close_all()`方法，去除`SSH.__del__()`方法
- 规范`SSH`的`with`上下文管理
- 创建`metaclass.py`用于实现测试协议的统一管理，测试具体产品时的配置基类
- 为`SSH`、`Channel`、`SFTP`类添加最近完成命令返回的record缓存属性`.last_record:CmdRecord[str]`
- `Bits`类添加切片置位方法`set_bits() __setitem__() `
- `Bits`类添分割方法`split() split_iter() ` 
- `Bits`类添加基础比较方法：`__eq__() __ne__() __le__() __lt__() __gt__() __ge__()`
- `Bits`类支持`bool`初始化
- `Bits`类添加添加`concat()`、`pop()`、`remove()`方法；`__setitem__()`支持了迭代赋值
- 添加`Bits._git_value()`
- `Bits`的`__getitem__()`与`__setitem__()`支持`[:]`全切
- 添加`BitView`类作为字段借用，与链接的`Bits`类同步数据
- 添加`Addr32`、`Addr64`类，是`Bits`类的简单封装与方法重载
- 添加`Field`类，主要作为`Register`类的附属
- 添加`Register`类，管理`Field`与`Addr32/Addr64`，可绑定读写寄存器的方法

## 0.1.2 修改

- `SSH.exec_run()`、`SSH.Channel.run()` 增加获取退出码功能 -> `CmdReocrd.exit_code`
- 添加 `SSH.long_running_list()`: 可迭代对象运行多个线程命令
- 添加 `SSH.exec_run_lines()` : 可迭代对象，顺序执行多条命令
- `SSH.long_running()`返回的`CmdRecording`实例添加了`pid`属性
- 添加`SSH.kill(pid)`方法，结束远程服务端进程
- `CmdRecording`添加`task_start()/task_join()/is_task_alive()`方法
- `SSH.exec_run_lines()`、`SSH.long_running_list()`和`Channel.run_lines()`方法添加了多行string去除纯注释行的功能
- 修复`SSH.exec_run()`输出可能获取不完整的bug
- 修改`CmdRecording.fifo_wait()`返回值从`tuple[Timestamp,Result[str]]`更改为`tuple[list[str],bool]`
- 添加`Obsidian`导出器，实现基本导出
- 修改`ulog`位值 `term.py`-> `control.py`
- 规范了一些范型代码
- 修复了`SSH.long_running()`无法自然结束的bug
- 添加了一些注释
- 实现了`Bits`类的多种格式化输出方式
- 基本实现了`Local`类基本的本地执行方法
- 补全了一些时间戳`Timestamp`的运算方法
- 修复了一些已知的问题
